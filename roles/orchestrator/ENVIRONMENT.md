# Environment

## Required Tools

Verify these are available before starting work:
- `git` — Version control
- `node` / `bun` — JavaScript runtime
- `agency` — Agency CLI

## Development Workflow

1. Create a branch for your work
2. Make changes
3. Run tests
4. Push and create a pull/merge request if applicable
5. Mark task as review
